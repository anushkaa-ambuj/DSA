| Metric               | Min-Heap (Priority Queue) | Ordered Map (std::map)  |
| -------------------- | ------------------------- | ----------------------- |
| **Time Complexity**  | **O(n log k)**            | O(n log n)              |
| **Space Complexity** | O(k)                      | **O(n)**                |
| **Speed (k ≪ n)**   | ✅ **Faster**             | ❌ Slower due to map ops |
| **Memory Usage**     | ✅ Lower                  | ❌ Higher                |
| **Stability**        | Good                      | Good                    |
